# TrueFoundry
 TrueFoundry Frontend Assignment - UI Schema to Form Generator

## Installation:
In the project directory, you can run the project locally:
1. Clone the repo.
2. npm run dev - Runs the app in the development mode.
3. Enter o to open the local host.
